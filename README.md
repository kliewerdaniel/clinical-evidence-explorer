# Clinical Evidence Explorer

Generated by the **Clinical Research Compiler** from a compiled
`clinical-knowledge-artifact.json`. This app is a *renderer* of the
artifact; the intelligence lives in the compiled evidence graph.

UI built with **Astryx** (@astryxdesign/core + theme-neutral), Next.js 15, React 19.

```bash
npm install
npm run dev
# open http://localhost:3000
```

Routes: overview, treatments, conflicts, guideline, sources.
